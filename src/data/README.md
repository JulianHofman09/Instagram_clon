# Carpeta `data`

Datos **locales** (no vienen de la API). Un archivo por tema:

| Archivo | Contenido |
|---------|-----------|
| `user.ts` | Usuario logueado simulado |
| `stories.ts` | Barra de historias |
| `suggestions.ts` | Sugerencias de la barra derecha |
| `postTexts.ts` | Usernames y captions del feed |
| `sampleComments.ts` | Comentarios del modal |
| `avatars.ts` | Helper `getAvatarUrl()` |

Las fotos de gatos se cargan en `services/catApi.ts`.
